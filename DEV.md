## Requirements
You need the following requirements:
 - Node 8+
 - NPM
 - React Native CLI
 - Jest
## Installation
```bash
npm install -g @codepso/rn-rad
npm update -g @codepso/rn-rad
```
## Test
```bash
yarn test
```
## Architecture
- React Native 0.6.1+
- React Navigation 5+
- React Native Elements 1.2.0+
- Vector Icons 6
- Redux
- Yup
- Formik
